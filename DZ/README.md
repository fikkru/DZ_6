# DZ_6
Шестое домашнее задание для самопроверки

Код для реализации функции trim_and_repeat(), которая принимает три параметра: 
● строку; 
● offset — число символов, на которое нужно обрезать строку слева; 
● repetitions — сколько раз нужно повторить строку перед возвратом получившейся строки. 

Число символов для среза по умолчанию равно 0, а количество повторений — 1. Функция возвращает полученную строку.
